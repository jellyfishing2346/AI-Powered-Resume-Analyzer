An intelligent system that automatically analyzes resumes, extracts key information using NLP techniques, and ranks candidates based on job requirements.

## Features

- **Resume Parsing**: Extracts text from PDF and DOCX files
- **Skill Extraction**: Identifies technical skills and competencies
- **Experience Analysis**: Calculates total years of experience
- **Education Detection**: Recognizes degrees and qualifications
- **Smart Ranking**: Ranks candidates based on job description matching
- **Customizable Weights**: Adjust importance of skills, experience, and education
